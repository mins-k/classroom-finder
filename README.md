## Classroom Finder (공강룸)

한남대학교 학생들을 위한 빈 강의실 조회 웹 서비스

공강 시간이나 팀 프로젝트, 자율 학습을 위한 공간을 쉽게 찾을 수 있도록 개발한 서비스입니다.

> 실제 학생들을 대상으로 운영되었으며, 지역 언론 기사에 소개된 프로젝트입니다.
> Backend 개발, 데이터베이스 설계 및 데이터 관리를 담당했습니다.

## 프로젝트 소개

대학교 학생들은 공강 시간에 공부하거나 팀 프로젝트를 진행할 공간이 필요하지만,
현재 사용 가능한 강의실을 확인하기 어렵다는 문제를 겪고 있습니다.

공강룸은 건물별 강의실 사용 현황을 제공하여 학생들이 빈 강의실을 빠르게 찾을 수 있도록 개발한 웹 서비스입니다.

##  담당 역할

### 김민석 (Backend)

- NestJS 기반 REST API 설계 및 개발
- MySQL 데이터베이스 설계
- 강의실 및 시간표 데이터 관리 로직 구현
- 데이터 수집 및 업데이트
- Backend 아키텍처 설계
## 기술 스택

### Backend

- NestJS
- TypeScript
- Node.js

### Frontend

- React
- JavaScript

### Database

- MySQL

### Infrastructure

- AWS EC2
- AWS RDS

## 성과

- 실제 학생 대상 서비스 운영
- 학교 커뮤니티 및 학과 사무실 홍보
- 지역 언론 기사 소개
